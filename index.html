Because WordPress is awesome, but the loop isn't.

Timber is a Wordpress theme that uses the Twig Templating Engine. This helps clean up your theme code so your single.php file can focus on the WordPress logic, while a single.html file can focus 100% on the HTML display.

Write better templates
By Jared Novack

Timber helps Upstatement write some very complicated WordPress sites very simply. Start using it today to create better, more maintainable code that can be shared and updated.

Less is more!

						<article>
							    <h1>{{ post.title }}</h1>
							    <small>{{ post.author }}</small>
							    <img src="{{ post.get_thumbnail.get_src }}" />
							    <p>{{ post.content }}</p>
						</article>

Good for template designers

Twig's beautiful syntax makes it easy to build your HTML and CSS without limit. It's easy to swap in new HTML for different sites or themes. That's less copy-and-pasting-and-rewriting and more extensibility.

Good for teams

Timber is built with teams of designers and developers in-mind. Not everyone knows the ins and outs of the_loop(), WordPress, and PHP. With Timber, your best WordPress dev can focus on the PHP, and your designers can work in much cleaner HTML markup.

Good for everybody!

Your new super-clean WordPress code will help you (and your clients) build-on the site for years to come.
